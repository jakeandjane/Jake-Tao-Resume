# Tao Youyang | AI Application / Agent / AI Infra Portfolio

This repository is my GitHub portfolio website for AI application development roles, with a focus on:

- Agent engineering
- LLM application development
- RAG and workflow automation
- AI infra / model deployment / evaluation

## Website structure

- `contents/home.md`: personal summary and role positioning
- `contents/projects.md`: selected projects
- `contents/experience.md`: internship and engineering practice
- `contents/skills.md`: technical stack and job fit
- `contents/config.yml`: site title and top banner text
- `index.html`: page structure
- `static/css/main.css`: customized styles
- `static/js/scripts.js`: dynamic content loading

## Deployment

This site can be deployed directly with GitHub Pages.

1. Push the repository to a GitHub repo named `<username>.github.io`
2. Enable GitHub Pages
3. Open the generated site URL

## Notes

This site is rewritten for job seeking rather than academic publication display.
